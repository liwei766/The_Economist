## 📖 关于本项目

欢迎来到 **《经济学人》周刊存档** —— 你获取《经济学人》杂志多种数字格式的综合资源库。本项目每周更新杂志内容，并配有深度博客文章，解析关键报道。

**为什么选择《经济学人》？**

- 🌍 **全球视野**：涵盖经济、商业、科技、文化和国际关系
- 📊 **专业分析**：深度报道和犀利评论，洞察世界格局
- 🧠 **知识沉淀**：阅读不仅获取信息，更提升批判性思维

---

## ✨ 特色功能

- 📚 **多种格式**：提供 EPUB、MOBI、AZW3 和 PDF 版本
- 🎧 **音频资源**：Google Drive 上的完整音频存档
- 📝 **每周博客解析**：深度分析封面故事和重点话题
- 🔄 **定期更新**：每周日上午 9:00（北京时间 UTC+8）
- 📂 **有序归档**：简洁清晰的周刊结构
- 🌐 **双语支持**：中英文文档

## 📁 仓库结构

```
The Economist/
├── TE-2025-01-04/           # 2025年1月4日刊
├── TE-2025-01-11/           # 2025年1月11日刊
├── TE-2025-01-18/           # 2025年1月18日刊
└── ...                      # 最新各期

每个周刊文件夹包含：
├── TE-YYYY-MM-DD.epub       # EPUB 格式
├── TE-YYYY-MM-DD.mobi       # MOBI 格式（旧版 Kindle）
├── TE-YYYY-MM-DD.azw3       # AZW3 格式（新版 Kindle）
├── TE-YYYY-MM-DD.pdf        # PDF 格式
└── TE-YYYY-MM-DD-音频/      # 音频文件夹（托管在 Google Drive）
```

> **注意**：历史存档（2020-2024年）存储在本地，不在此仓库中跟踪。

## 📖 可用格式

| 格式 | 最适合 | 设备兼容性 |
|------|--------|-----------|
| 📘 **EPUB** | 通用电子阅读器 | 大多数电子阅读器、平板、手机 |
| 📙 **MOBI** | 旧版 Kindle 设备 | Kindle Paperwhite（2022年前）、Kindle Keyboard |
| 📕 **AZW3** | 新版 Kindle 设备 | Kindle Paperwhite（2022+）、Kindle Oasis |
| 📄 **PDF** | 打印、大屏幕 | 电脑、平板、高分辨率显示器 |
